# Tower Defense: Factory Under Siege

A Tower Defense mod for [Satisfactory](https://www.satisfactorygame.com/) — Defend your factory against waves of hostile creatures!

## Features

- **Laser Turret** — Long-range precision beam, requires power
- **Fire Turret** — Rapid-fire bullets, ammo-based, no power needed
- **Tesla Shield Generator** — Protective energy dome with rotating Tesla rings that shields your buildings
- **Shockwave Turret** — Devastating ground pound that knocks back groups
- **Drone Platform** — Automated ammo delivery drone for your fire turrets
- **3 enemy types** — Ground, Ram (wall breaker), and Flying
- **Night wave system** — Difficulty scales with your factory size
- **Smart enemy AI** — Enemies analyze your base, find weak points, and prioritize targets
- **Fully automated** — Turrets auto-target, drones auto-deliver, waves auto-trigger at night

## Installation

1. Install [Satisfactory Mod Manager (SMM)](https://smm.satisfactorymodding.com/)
2. Search for **"Tower Defense"** in the mod manager
3. Click install

Or manually:
1. Download the latest release from [ficsit.app](https://ficsit.app)
2. Extract the zip into `Satisfactory/FactoryGame/Mods/`
3. Make sure the path is `Mods/MonPremierMod/MonPremierMod.uplugin`

## Requirements

- Satisfactory (Steam)
- SML (Satisfactory Mod Loader) ^3.0.0

## Alpha Disclaimer

This mod is in early alpha. There may be some lag spikes at the start of each wave (terrain scanning & pathfinding), and a few bugs are likely still present. This is a solo passion project and I'm releasing it early to get community feedback. If you enjoy the concept, let me know — your feedback will decide whether I keep developing this mod or not! Bug reports, suggestions, and ideas are all welcome.

## Building from source

1. Clone the [SatisfactoryModLoader](https://github.com/satisfactorymodding/SatisfactoryModLoader) repository
2. Copy the `MonPremierMod` folder into `Mods/`
3. Build with Unreal Engine (CSS fork) in Shipping configuration
